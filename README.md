# randget
Android Budget Tracker app for personal finance management

## 📱 Overview

**RANDGET** (Rand + Budget) is a comprehensive personal finance management application for Android. It helps South African users track expenses, set budget goals, and visualize spending patterns to make every rand count.

## 🎯 Features

| Feature | Description |
|---------|-------------|
| 🔐 **User Authentication** | Secure login and registration system |
| 💰 **Budget Management** | Set monthly budget using interactive SeekBar |
| 📝 **Expense Tracking** | Add expenses with name, amount, category, and date |
| 📊 **Statistics** | View spending breakdown with interactive PieChart |
| 🗂️ **15+ Categories** | Pre-defined categories with emoji icons (Food, Transport, Bills, etc.) |
| 💾 **Local Database** | Room database for persistent data storage |
